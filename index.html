Xtheme IRC Services

Xtheme is a set of services for IRC networks designed for large IRC networks with high scalability requirements. Xtheme is feature-packed, fast, efficient, customisable, extendable and ubiquitous. It is relatively mature software, with some code and design derived from other packages called Atheme and Shrike.

SWI (Web Panel)

SWI (Services Web Interface) a Web Panel for IRC Networks running Xtheme IRC Services / Atheme IRC Services (or Atheme forks) that allows users complete management of their accounts, channels, memos, vhosts, and groups. Network staff can also manage Services functions and akills via the Web Interface.
